About

Pi-Lock is an automated door Security System built around the Raspberry Pi. Using a RFID reader and PIN entry system, access to buildings, departments and rooms is controlled in a secure, efficient, and cost-minimizing system.

Built for Any Application See for yourself.

Pi Lock can be used for any application and product. Whether you want to use it for a door security system, or as a Kiosk to manage employee expenditure in the company shop, or to simply open your garage door. Jump into the source code to find out what you want to use it for.

Add Extra Components done.

With Pi Lock you can add any Arduino or Raspberry Pi compatible sensor or component. Need a Temperature and Light reader? Done. Motion detector? Done. Just use the one of the component classes in the github project and tailor it to your device, and it will be working in a jiffy.
